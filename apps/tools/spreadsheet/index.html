Cheat Sheet & Shortcuts

Supported Formulas

Start cell input with = to create calculations. Case insensitive. Ranges use colon :.

=SUM(A1:B3) Adds all values in cells A1 through B3.
=AVERAGE(C1:C10) Calculates mathematical average of the column range.
=COUNT(A1:D1) Counts numeric entries in the specified row range.
=MIN(B1:B5) / =MAX(B1:B5) Finds the minimum or maximum value in the range.
Basic Math: =A1+B2*3 - C4/2 Performs cell arithmetic operations supporting addition (+), subtraction (-), multiplication (*), division (/), and numbers.

Classroom Tips

Double-click headers (A, B, C or 1, 2, 3) to resize column widths or row heights if your vocabulary or titles are extra long. Pressing F11 makes the spreadsheet go full-screen on your classroom projector.
